Static Classes, and Math.random()

Math is a class that is built in to Java. The "M" is capitalized because, in Java, capitalizing the first letter of a class follows standard naming convention. It is a "final static" class, which will be discussed later in this post. Information on all of the methods in the class Math can be found by clicking here.

I mentioned that "Math" is a static class, or a class that cannot be called using a reference variable. This is because it has a constructor that was declared "private". A constructor is added automatically by the compiler if the programmer doesn't declare one. A public constructor would look like this, public Math(){}, and a private constructor would look this, private Math(){}. A private constructor means that outside of that class, the constructor cannot be called. Therefore, an object cannot be made of that class. The examples below show the difference between a public class and a static class:

Public Class Example: 
public class NonStaticExample{
  public NonStaticExample(){
    //This is a constructor in its default state
    //the compiler will do this automatically if
    //the programmer doesn't put a constructor here
  }

  static void printMe(String thingToPrint){
    System.out.println(thingToPrint);
  }
}
Static Class Example: 
public class StaticExample{
  private StaticExample(){
    //This is the constructor
    //Since it is marked private,
    //an ojbect cannot be made of
    //this class.
  }

  static void printMe(String thingToPrint){

    System.out.println(thingToPrint);
  }
}

As you can see in the examples, the static class's constructor is private StaticExample(){}, and the public class's constructor is public NonStaticExample(){}. The "private" disallows any instantiation outside of the class.

Public Object Example (Makes an object of NonStaticExample from the Public Class Example): 
public class NonStaticObject{
  public static void main(String[] args){

    //now we can make an instance variable call the method.
    //if, in NonStaticExample.java, the constructor was
    //marked private, this code would not compile.
    NonStaticExample exampleObject = new NonStaticExample();
    exampleObject.printMe("This Will Print");
  }
}
Static Object Example (Cannot make an object of type StaticExample from the Static Class Example): 
public class StaticObject{

  public static void main(String[] args){

    String myString = "Print This!";

    StaticExample.printMe(myString);

    //This file will not compile if
    //the statement below were included, because
    //StaticExample is marked private,
    //therefore a new Object cannot be instantiated.

    //StaticExample newObject = new StaticExample();

  }

}

The only way to use a method in the static class, is to use the following convention: ClassName.method(). In a public class, as seen in the example, it is possible to make a reference variable referencing an object of that type, and then calling a method on that reference variable.

Since the "Math" class in Java is static, an object of type "Math" cannot be assigned to a reference variable, and its methods must be called in the following manner: Math.random().

Random Method Example: 

public class RandomMath{
  public static void main(String[] args){
    double x = Math.random();
    System.out.print("Random Stuff: ");
    System.out.println(x);
  }
}

".random()" is a method of the class "Math". The ".random()" method's return type is double. Specifically, it returns a double value from zero, up to (but not including) one. If the code above is ran, "x" will be assigned a random value greater or equal to zero, but less than one.

Math is also declared as a final class, which means it cannot be extended.

Extending a Non-Final Class Example: 
public class Extending extends NonStaticExample{

  public static void main(String[] args){
    //this has the capabilities from NonStaticExample
    //but we are also able to add new things
    //it can be extended since it is not a Static class
    //It would be impossible to try to extend
    //the class "StaticExample," since it is private.
    printMe("This is a method from NonStaticExample!");
    System.out.println("This is new stuff!");
  }
}
Final Class Example: 
final class NonStaticExampleFinal{
  public NonStaticExampleFinal(){
  }

  static void printMe(String thingToPrint){
    System.out.println(thingToPrint);
  }
}
Extending a Final Class Example: (DOES NOT COMPILE (on purpose...)) 
public class ExtendingFinal extends NonStaticExampleFinal{
  //**************************************************
  //this fails to compile, since NonStaticExampleFinal
  //is a final class.
  //***************************************************

  public static void main(String[] args){

    printMe("This is a method from NonStaticExample!");
    System.out.println("This is new stuff!");
  }
}
