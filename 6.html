Parameter vs. Argument

Parameters and arguments must be used together to pass values into a method. A parameter is something that is written with a method when the method is declared. It states what types of values that the method will be accepting, and variable names for those values. An Argument is something that is passed into the method. For example, in the code below, the parameter is and integer variable named "y". The argument (the value passed to the method "printStuff") is "x". The value in argument "x" is then stored in the parameter "y" for use within the "printStuff" method.

Parameter vs Argument Example: 
public static void main(String[] args){

  int x = 2;
  printStuff(x); //"x" is the argument
}

//Below, "y" is the parameter.  It stores
//the value from the argument "x" for
//use wthin the "printStuff" method.
public static void printStuff(int y){
  System.out.println(y);
}
