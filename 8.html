Imports, java.util.Scanner, and java.lang

When "java.util.Scanner" is imported into the code, it allows Java to use the "Scanner" class from the "java.util" package without telling the computer that Scanner is in the "java.util" package each time it must be used. The "java" in "java.util.Scanner" means it is from the "java" package, and "java.util" is a sub-package. If "java.util.Scanner" is not imported, then the "Scanner" class can still be used, however, it must be typed out each time. The examples below display the difference in making a "Scanner" object with and without the import.

Java.util.Scanner Imported: 
import java.util.Scanner;

public class ScannerImport{
  public static void main(String[] args){
    Scanner keyboard = new Scanner(System.in);
    System.out.print("Write something, and it will print: ");
    String toPrint = keyboard.nextLine();
    System.out.println(toPrint);
  }
}
Java.util.Scanner Not Imported: 
public class ScannerNoImport{
  public static void main(String[] args){
    java.util.Scanner keyboard = new java.util.Scanner(System.in);
    //"java.util" must be used before "Scanner"
    System.out.print("Write something, and it will print: ");
    String toPrint = keyboard.nextLine();
    System.out.println(toPrint);
  }
          }

Another good example is the "java.lang.System" class. It is used often for printing. Normally, a print statement would look like this: System.out.println("Stuff to print");. In Java, "java.lang" is automatically included, so it is possible to write "System.out.println()" without either typing it all out as "java.lang.System.out.println()", and without importing "java.lang". It is possible to import "System", however, reducing the necessary statement to "out.println()". The examples below show normal printing, and printing with the "java.lang.System.*" import.

Printing Without java.lang.System.* imported: 
// java.lang is already included.

public class PrintNormal{
  public static void main(String[] args){
    System.out.println("Printing stuff.");
  }
}
Printing With java.lang.System.* imported: 
import static java.lang.System.*;

public class PrintImport{
  public static void main(String[] args){
    out.println("Printing stuff.");
  }
}
