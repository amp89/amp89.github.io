Casting

Casting a double variable to an integer variable removes everything after the decimal place. Given a double x and an integer y, the code to store x's value in y would look like this: y = (int)x;.

Casting Example: 

double x = 4.564654;
int y;

//the "(int)" changes the x to an integer, a
//value that can be held by y, by dropping
//everything after the decimal point
y = (int)x;

One purpose of casting a double to an int is to generate a random integer. Java's Math.random() can only produce a number greater to equal to zero and less than one. In order to create a random integer, we have to modify that number by multiplying it, and casting it to an int. The statement to generate a random number between 0 and 5 would look like this: int x = (int)(Math.random()*6);. Multiplying Math.random() by 6 would return any double from 0 to 5.99999â€¦, and casting any of these numbers to an integer would return any integer from 0 to 5.

Casting Random Number Example: 
//create a random integer between 0 and 5;
int x = (int)(Math.random()*6);
