Three Types of Equal

There are 3 types of equal operators in Java (=, ==, and .equals).

The first one, "=", simply assigns values to a variable. The value on the right side of the operator is assigned to the variable on the left.

Single Equal Example: 
int x = 2; //The = assigns the value 2 to x.
String y = "Stuff"; //The = assigns "Stuff" to y.

The "==" is a little different. It compares to numerical values, but should not be used to compare strings. If the values on both sides of "==" are the same, then it will be the same as the boolean value true. For example, if there were an integer variable "x" that was equal to "2" then the following would be true: Writing if (x=2) is the same as writing if (true).

Double Equals Example: 
int x = 2;

if (x==2);
//the == compares the variable x to the integer 2
{
	System.out.println("X must equal 2 for this to print.");
}

Last, the ".equals" is used to compare two strings. If the string or string variable before the "." is the same as the string or string variable in the parenthesis, then it will be true. For example, if there was a string variable "x" that was equal to "Hello!", the following would be true: Writing if (x.equals("Hello!")) is the same as writing if(true).

".equals" Example: 
String x = "Hello!";

//the .equals compares string values, the same way ==
//compares numerical values
if (x.equals("Hello!")){  compares numerical values.
  System.out.println("x must be equal to \"Hello!\"");
}

if ("Hello!".equals(x)){
  System.out.println("This works too!");
}

if ("Hi!".equals("Hi!")){
  System.out.println("And so does this!");
}
